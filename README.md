# weeek
A really simple web-based weekly planner

## Runs at

[http://weeek.surge.sh/](http://weeek.surge.sh/)

## Usage

Run locally:

```bash
$ yarn start
```

Build for production:

```bash
$ yarn run build
```

Deploy to weeek.surge.sh (need to be signed in as me):

```bash
$ yarn run deploy
```
